How it works

One controlled workflow from idea to post.

Goldmine Agent is built for the Goldmine Trading education brand. It keeps the creator in control while removing repetitive rendering, upload, and publishing steps.

Connect TikTok

The user signs in with TikTok Login Kit and authorizes the requested account permissions.

Create video

The tool renders a 9:16 educational trading video from approved Goldmine content and brand assets.

Review content

The user checks the video, caption, and educational disclaimer before anything is uploaded.

Upload or publish

The approved video is sent to TikTok as a draft or published directly through the Content Posting API.

TikTok integration

Scopes used by Goldmine Agent.

The TikTok permissions requested by this app are limited to account verification and video posting actions selected by the user.

Login Kit

  • user.info.basic verifies the connected TikTok account after OAuth.
  • Redirect URI returns users to the Goldmine Agent callback page after authorization.
  • Account control stays with the user. The user can revoke access from TikTok settings.

Content Posting API

  • video.upload uploads approved MP4 content to TikTok as a creator draft.
  • video.publish publishes a reviewed video when direct posting is selected.
  • No scraping or unrelated TikTok data collection is performed by Goldmine Agent.
Safety and compliance

Human review stays in the loop.

Goldmine Agent is not financial advice software. It is a content workflow tool for educational trading media. Videos are reviewed by the Goldmine Trading team before upload or publishing, and posted content must comply with TikTok policies, platform terms, and applicable financial promotion rules.

OAuth tokens are used only to complete the user-requested TikTok posting workflow. See the Privacy Policy and Terms of Service for the full details.
